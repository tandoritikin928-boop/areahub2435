Executors

実行環境・安定性・検出耐性を軸に整理された Executor 情報。
表面的な評価ではなく、構造と挙動を理解するための資料として掲載。

🔗 Xeno Executor（PC） ＋
https://www.xeno.onl/

Xeno は Windows 環境向けに設計された比較的軽量な Executor で、 DLL インジェクション方式を採用している。起動速度が速く、 UI も簡素なため初心者でも扱いやすい一方、 Roblox 側の更新に影響を受けやすい特性を持つ。 実行時はロード順や API 呼び出しのタイミングが重要で、 loadstring 実行前後の挙動を観察することで ゲーム側の制御構造を把握しやすい。 研究用途では、検出ポイントや失敗時の挙動解析に向いている。

🔗 Potassium（PC / 有料） ＋
https://rscripts.net/executor/potassium

Potassium は有料モデルを採用した PC 向け Executor で、 安定性と更新頻度を重視した構成が特徴。 API フック処理が比較的安定しており、 複雑な UI スクリプトや長時間実行にも耐えやすい。 実験用途では、安定環境下でのスクリプト挙動確認や パフォーマンス差分の比較に向いている。 有料である分、サポートと更新対応が早い点が利点。

🔗 Solara（PC） ＋
https://getsolara.dev/

Solara は比較的新しい Executor で、 シンプルな UI と高速な初期化処理を特徴とする。 実行構造が分かりやすく、API 呼び出しの流れを 追いやすいため解析向け。 実験では、短いスクリプトの実行挙動や Roblox 更新直後の影響確認などに使われることが多い。

🔗 Delta Executor（iOS / Android） ＋
https://delta-executor.com/

Delta はモバイル環境向けに設計された Executor。 iOS / Android 双方に対応し、ipa や apk を通じて動作する。 PC Executor と異なり、 実行制限や API の一部が制限されるため、 環境差による挙動比較や制限下での設計研究に適している。 モバイル特有の制御構造を理解する目的で利用される。
