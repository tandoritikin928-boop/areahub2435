Scripts Library

研究・検証用途として集約されたスクリプトカテゴリ。
構造を理解し、挙動を観察するための情報群。
表に出ない仕組みを、静かに読み解く。

ScriptBlox

ScriptBlox は Roblox スクリプトをカテゴリ・人気度・API構造ごとに閲覧できる巨大データベース。
loadstring 構文を中心に、RemoteEvent / RemoteFunction の使い方、
Character 制御、Humanoid 操作、UI生成などの実装例が豊富に蓄積されている。

公開スクリプトはそのまま使うためではなく、
「どの API が、どの順で呼ばれているか」を読むことで、
ゲーム側の構造や制御ポイントを理解するための資料として価値がある。

Autofarm ＋

ゲーム内の報酬・進行処理を自動化するカテゴリ。
ループ構造、待機処理、イベント監視の実装研究向け。

Kill Aura ＋

周囲のエンティティ検出・距離判定・攻撃トリガーの制御を確認するカテゴリ。
判定条件や最適化の考え方を学ぶための研究用。

Utility ＋

UI補助、ステータス表示、操作簡略化などの便利系スクリプト。
APIの呼び方や安全なフック方法の理解に向いている。

Fly / Movement ＋

Character の座標操作、BodyVelocity / CFrame 制御の研究カテゴリ。
移動制御ロジックの理解に使われる。

AFK / Idle ＋

ユーザー状態監視、一定間隔での入力送信など、
ゲーム内アイドル検知回避の仕組みを学ぶカテゴリ。
